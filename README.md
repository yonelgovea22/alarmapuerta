# alarmapuerta

<h1>Alarma de Puerta </h1>


<h2>Material:</h2><br>

<li>Arduino </li>
<li>Proto  </li>
<li>Reed switch</li>
<li>Cables </li>
<li>Buzzer</li>
<li>Led´s</li>
<li>Resistencias </li>
<li>Librería para lainterrupcion del  Arduino </li><br><br>

reed switch, logrando disminuir el consumo de energía.
